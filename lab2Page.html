Lab 2 - Semantic Wikipedia Page

The first lab for this class was all about creating a HTML5 semantic version of any webpage on Wikipedia. For this lab, I chose to do the Wikipedia page about City of Bones, a novel by Cassandra Clare. Prior to this lab, we learned techniques about having a navigation at the top, having a main element and then sections within the main to differentiate content. I learned a lot about the basics of accessibility techniques and implemented them in this lab.
